#  Data Gathering in Python

This project focuses on practicing **data collection techniques**, which form the foundation of any data science workflow.  
The notebook demonstrates how to gather data from different sources such as JSON files and public APIs, and convert them into structured tabular formats for analysis.

---

<!-- =============================================== -->
<!--                ABOUT THE PROJECT                -->
<!-- =============================================== -->

##  About the Project

In this notebook, I explored multiple approaches to collect and prepare raw data using Python, including:

- Working with flat JSON files  
- Handling nested JSON structures  
- Normalizing JSON into tabular format using pandas  
- Fetching real-time data from an external API (CoinGecko)  
- Converting API responses into pandas DataFrames  

The goal is to understand how data is gathered before performing cleaning, EDA, or modeling.

##  Topics Covered

- Reading and writing JSON files in Python  
- Parsing flat vs nested JSON data  
- Using `pd.json_normalize()` for flattening data  
- Making API calls using `requests`  
- Transforming API responses into DataFrames  
- Basic inspection of collected data
